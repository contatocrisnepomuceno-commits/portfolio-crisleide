# 🌟 Portfólio — Crisleide Silva Nepomuceno

Portfólio profissional em formato fanpage HTML estático, pronto para publicação no GitHub Pages.

## 🚀 Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `portfolio-crisleide`)
2. Faça upload do arquivo `index.html` para a raiz do repositório
3. Acesse **Settings → Pages**
4. Em **Source**, selecione `main` branch e pasta `/ (root)`
5. Clique em **Save**
6. Seu portfólio estará disponível em: `https://SEU_USUARIO.github.io/portfolio-crisleide`

## 📁 Estrutura

```
portfolio/
└── index.html    ← Portfólio completo (tudo em um único arquivo)
└── README.md     ← Este arquivo
```
